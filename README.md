# BookingHub Frontend

Vue.js 3 frontend for BookingHub — the first unified booking platform in Kyrgyzstan.

## Live Demo
https://booking-hub-frontend-eight.vercel.app

## Tech Stack
- Vue.js 3 (Composition API)
- Pinia (state management)
- Vue Router
- Tailwind CSS
- Axios
- Leaflet.js (interactive map)
- vue-i18n (EN, RU, KY)
- OpenRouter AI API

## Installation
```bash
npm install
cp .env.example .env
npm run dev
```
